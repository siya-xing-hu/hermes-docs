## 🔄 工作流程

1. **自动记录**：每次会话结束后，AI 自动整理工作内容并提交
2. **定期反思**：每周生成一次反思报告，总结做得好的和需要改进的
3. **用户审阅**：用户定时查看记录，给出建议
4. **规范更新**：AI 拉取用户建议，更新 `guidelines/` 下的规范文件
